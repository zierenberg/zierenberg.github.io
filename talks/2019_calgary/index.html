Model: Branching network with homeostatic self-regulation
  • time step $\Delta t$
  • external excitation: rate $h$
  • internal excitation: connectivity matrix $w_{ij}$
  • spiking neuron excites on average $m=f(w_{ij})$ further neurons
  • key parameters:
    • $h$ strength of external input
    • $m$ strength of internal activation

post-synaptic weight $\alpha_j$

$\Delta\alpha_j = (\Delta t~r^\ast - a_j)\frac{\Delta t}{\tau_\mathrm{hp}} \qquad\text{with}\qquad \alpha_j\geq0$

Emergent temporal correlations in self-regulated networks

Mean-field theory:
  • stationary activity in a branching process

    $\mathbb{E}(A_{t+1}|A_t) = m A_t + Nh\Delta t \hspace{3em}$

    with $\overline{a} = \frac{A}{N} = \frac{h \Delta t}{1-m}$

  • homeostatic regulation

    with $\overline{a} \to \Delta t~r^\ast$

Equating both results, we obtain for the branching parameter

$m = 1-h/r^\ast$

The branching parameter is directly related to an exponential autocorrelation time

$\tau = -\Delta t/\ln(m) = -\Delta t/\ln(1-h/r^\ast)$

Inferring autocorrelation time of subsampled systems with non-stationary external input

Observation: Neuroscience experiments assume consistent responses across trials.
Assumption: Internal spreading dynamics are stable, but external input changes reliably.
Pipa et al., Front. Integr. Neurosci. (2009)
Model:
Subsampled spreading process with an autoregressive representation

$\hspace{3em}\mathbb{E}(A_{t+1}|A_t) = m A_t + \mathbb{E}(h_t)\hspace{1em}$ with $\hspace{1em}\mathbb{E}(a_{t}|A_t)=\alpha A_t $

Generalization to multiple steps

$\hspace{3em}\mathbb{E}(A_{t+k}|A_t) = m^k A_t +\sum_{l=1}^{k} m^{k-l} \cdot \mathbb{E}(h_{t+l-1})$

For stationary external input one can show a constant bias in

$\hspace{3em}C(k)=\frac{\mathrm{Cov}(a_t,a_{t+k})}{\mathrm{Var}(a_t)} = bm^k=be^{-k\Delta t/\tau}$

This is recovered for mean-corrected activity $\tilde{a}_t^i = a_t^i - \langle a_t\rangle$

$\hspace{3em}b_t=1/(1-(1-\alpha^{-1})F_t^{-1})$

Wilting & Priesemann, Nat. Commun. (2018)
